# Singly Linked-List
A linked list base implementation of List in C++.

**Methods Included**

1. Add the elements into the lists
2. Remove the elements by its position
3. Remove the elements by its value
4. Clear the list
5. Check whether list is empty or not

***To run the code:***

*g++ List.cpp Main.cpp*


***To excute the code:***

*.\a.exe*
